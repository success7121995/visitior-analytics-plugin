Track and visualize visitor data with an interactive dashboard in WordPress.

== Description ==

WP Visitor Analytics is a powerful WordPress plugin that helps you track and analyze visitor data on your website. It provides detailed insights into your website's traffic through an easy-to-use dashboard with interactive charts and statistics.

= Key Features =

* **Visitor Tracking**
  * Track total number of visitors
  * Monitor unique visitors by IP address
  * Record visitor's device and browser information
  * Track visitor's geographical location
  * Monitor landing pages and referrers

* **Interactive Dashboard**
  * Monthly visitor statistics with bar charts
  * Daily visitor breakdown
  * Year and month selection for data filtering
  * Responsive design for all devices

* **Geographical Analytics**
  * Country-based visitor statistics
  * City and region tracking
  * Network and ISP information
  * Timezone tracking

* **Technical Insights**
  * Browser usage statistics
  * Device type analysis
  * Network information
  * Language preferences

* **Data Privacy**
  * Cookie consent management
  * GDPR compliant tracking
  * Customizable tracking preferences
  * Data retention controls

== Installation ==

1. Upload the `visitor-analytics` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Visitor Analytics' in your WordPress admin menu to access the dashboard

== Frequently Asked Questions ==

= Does this plugin comply with GDPR? =

Yes, the plugin includes cookie consent management and allows visitors to control their tracking preferences.

= How is visitor data stored? =

Visitor data is stored in the WordPress database in a dedicated table. The plugin creates a custom table named `wp_visitor_analytics` upon activation.

= Can I customize the tracking preferences? =

Yes, you can manage tracking preferences through the plugin's settings page, including which data points to track and store.

== Privacy Policy ==

This plugin collects and stores visitor data including:
* IP address
* Browser information
* Device type
* Geographical location
* Visit time
* Landing page
* Referrer information

The data is used solely for analytics purposes and is stored in your WordPress database. You are responsible for informing your visitors about the data collection and ensuring compliance with privacy laws.

== Cookie Policy ==

The plugin uses cookies to:
* Track visitor preferences
* Store consent settings
* Maintain session information

Cookies are set with a one-year expiration date and can be managed through the plugin's settings.

== Technical Requirements ==

* WordPress 5.0 or higher
* PHP 7.0 or higher
* MySQL 5.6 or higher
* JavaScript enabled in the browser
