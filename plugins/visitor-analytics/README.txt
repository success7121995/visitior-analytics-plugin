== Description ==

Visitor Tracker: Analytics for WP is a powerful WordPress plugin that helps you track and analyze your website visitors. It provides detailed insights about your visitors' behavior, including their location, browser, device, and more.

= Key Features =

* Real-time visitor tracking
* Beautiful and interactive charts
* Detailed visitor statistics
* Export data to CSV and JSON formats
* REST API endpoints for data access
* Responsive dashboard design
* Customizable date ranges
* Export functionality for all data

= Visitor Information Tracked =

* Total visitors count
* Visitors by month
* Visitors by day
* Visitors by IP address
* Visitors by country
* Visitors by browser
* Visitors by device
* Visitors by city
* Landing page statistics

= Charts and Visualizations =

* Monthly visitor trends
* Daily visitor patterns
* Interactive bar charts
* Responsive design
* Customizable date ranges
* Export capabilities

= Data Export =

* Export to CSV format
* Export to JSON format
* Export specific time periods
* Export individual statistics
* Download full data sets

= REST API =

The plugin provides REST API endpoints for accessing visitor data:

* `/wp-json/visitor-analytics/v1/monthly-data` - Get monthly visitor data
* `/wp-json/visitor-analytics/v1/daily-data` - Get daily visitor data

== Installation ==

1. Upload the `visitor-analytics` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Visitor Analytics' in your WordPress admin menu
4. Start tracking your visitors!

== Frequently Asked Questions ==

= How does the plugin track visitors? =

The plugin tracks visitors using a combination of IP address tracking and browser fingerprinting to ensure accurate visitor counts while respecting privacy.

= Does this plugin affect website performance? =

The plugin is optimized for performance and has minimal impact on your website's loading speed. All heavy processing is done asynchronously.

= Can I export the data? =

Yes, you can export all visitor data in both CSV and JSON formats. The export functionality is available for all statistics and charts.

= Is there an API available? =

Yes, the plugin provides REST API endpoints for accessing visitor data programmatically. API documentation is available in the plugin's documentation.

= How long is the visitor data stored? =

Visitor data is stored in your WordPress database. You can manage the data retention period through the plugin settings.

== Privacy Policy ==

WP Visitor Analytics collects visitor data to provide analytics and insights. This data includes:

* IP addresses (anonymized)
* Browser information
* Device information
* Geographic location
* Visit timestamps
* Landing pages

This data is used solely for analytics purposes and is not shared with third parties. Users can export and delete their data at any time through the plugin's interface.
